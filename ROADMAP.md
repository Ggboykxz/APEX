# APEX Public Roadmap

## ✅ v1.3.0 — Foundation (released May 8, 2026)
- 100+ model support via litellm
- Multi-agent system (Build, Plan, Explore, General)
- 75+ built-in tools
- Beautiful TUI (Textual 8)
- `pip install apex-agent`
- MIT license
- Session persistence
- Plugin system
- Command palette
- Auto-completion

## 🔧 v1.4.0 — Power (Next)
- Streaming responses (real-time output)
- Git tools (status/log/diff/commit/branch)
- Web search tool
- Session save/load improvements
- `/memory` persistent context

## 🎯 v2.0.0 — Intelligence
- Repo map (understand full codebase structure)
- Vision support (send screenshots to vision models)
- Auto-commit mode
- VS Code extension
- MCP server mode

## 🌍 v2.5.0 — Enterprise
- Full test suite (>80% coverage)
- Windows installer (.exe)
- macOS app wrapper
- Enterprise features (SSO, audit logs)
- APEX Cloud (hosted version)
