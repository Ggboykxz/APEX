Features

Everything You Need

APEX combines the power of 170+ models, 75+ tools, 5 specialized agents, and OpenCode architecture in one terminal-native experience.

170+ Models

Use any LLM from any provider. GPT-5, Claude Sonnet 4.6, Gemini 3, Grok 4, DeepSeek V4, Qwen 3.6, and 150+ more models via litellm.

11 Specialized Agents

4 Primary agents (Coder, Architect, Planner, Shell), 4 Subagents (@reviewer, @general, @explore, @scout), and 3 system agents. Full primary + subagent system with @mention support.

75+ Tools

File ops, search, git, web, LSP, code generation, sandboxed execution, clipboard, skills, and more — all built in and ready.

Security System

Shell command analysis, permission rulesets (ALLOW/DENY/ASK), rate limiting, API key management, billing system, and path traversal protection.

Switch Models Live

Switch between any model mid-session without restarting. Compare outputs, optimize costs, and never lose context.

Ink TUI + WebSocket EventBus + 12 Themes

Rich Ink + React TUI with WebSocket EventBus for real-time sync, command palette (Ctrl+P), leader keys (Ctrl+X), @ file references, ! bash inline, 12 built-in themes with dark/light mode. OpenCode architecture with random port binding.

Snapshots & Undo/Redo

Git-based snapshot system creates automatic backups before every destructive action. Undo/redo with full diff computation. Never lose work again.

Custom Commands + Session Sharing + Formatters

Create custom commands via .md files with $ARGUMENTS, !shell, @file. Share sessions via public URLs. 11 auto-formatters (ruff, prettier, gofmt, rustfmt, etc.). File watcher with gitignore-aware patterns.

What it actually is

01 · Terminal Agent

A coding agent, not a chat box

Same loop as Claude Code or Codex CLI. It reads, edits, runs tests, reports back.

02 · Sandbox Protection

Five agents, one approval system

Coder asks, Architect reads, Planner observes. Sandboxed via landlock (Linux), seatbelt (macOS), restricted tokens (Windows).

03 · Model Freedom

170+ models by default

Use any LLM from any provider. Swap with /model command mid-session. Structured messages with Parts, snapshots with undo/redo, and custom commands.
